# 現代極簡風電風扇網頁

一個基於 HTML5/CSS3/Vanilla JS (ES Modules) 實作的純前端極簡風電風扇，具備逼真的起步加速與停轉慣性物理模擬。

## 特色
- **極簡美學**：無印良品（MUJI）啞光白風格，使用純 CSS 向量與 HSL 顏色系統繪製電風扇。
- **物理慣性模擬**：啟動時由靜止逐漸加速至最高速；關閉時隨空氣阻力與摩擦力緩緩減速至靜止。
- **風流粒子動畫**：開啟時兩側會出現流線型風吹動效，增強視覺互動感。
- **零依賴**：無須任何第三方前端套件，極致輕量。
- **部署友善**：專門為 GitHub Pages 設計，隨開即用。

## 本地測試與驗證
本專案使用 Node.js 原生測試框架對物理狀態模型進行驗證：
```bash
# 執行單元測試
npm test
```

## 部署到 GitHub Pages
1. 將此專案推送至您的 GitHub 儲存庫。
2. 進入儲存庫的 **Settings** -> **Pages**。
3. 將 **Source** 設為 `Deploy from a branch`。
4. 選擇您的主要分支（如 `main`）並指定根目錄 `/`。
5. 點擊 **Save**，稍等幾分鐘即可完成部署！
